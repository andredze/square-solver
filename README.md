## KVADRATKA
### Made on LETKA by KSA

Build:
```powershell
  make
```
Run with data from keyboard:
```powershell
  run.exe
```

To run programm with data from file, make file with such text structure:
```text
    a1 b1 c1
    a2 b2 c2
    ...
```

Run with data from file:
```powershell
  run.exe <your_file_name>
```
